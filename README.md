# Payloads
1. Swappable payload bays
- Cameras
- LiDAR
- Thermal/IR sensors
- Multispectral imagers
- Speakers or lights
- Delivery mechanisms

2. Interchangeable flight stacks
- Flight controller
- GPS/RTK modules
- Communications (4G/5G, LoRa, long-range radios)

3. Mission-specific add-ons
- Obstacle avoidance
- Extended battery packs
- Docking/charging modules

4. Foldable or expandable arms and frames
Useful for transport + different sizes of payloads

# Repository Structure
- Hardware: stores all pcb designs and 3d print files for simpler production 
- Firmware: loaded via UART to main controller at step 3
- Software: computer downloaded software for GUI and controlling of aircraft
